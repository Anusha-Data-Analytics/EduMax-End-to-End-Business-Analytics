# Dashboards

This folder contains the interactive Power BI dashboards developed as part of the EduMax Business Analytics project.

Each dashboard is designed to support stakeholder decision-making by presenting key business metrics, performance trends, and analytical insights in a clear and actionable format.

## Available Dashboards

Monitor revenue performance, evaluate revenue trends, and support executive decision-making through interactive visual analytics.
