# EduMax : End-to-End Business Analytics

## End-to-End Business Analytics Portfolio Project

**Domain:** EdTech Analytics

**Tech Stack:** SQL • Python • Power BI

## Repository Identity

A consulting-style end-to-end Business Analytics project demonstrating how data-driven insights can improve learner engagement, business performance, and strategic decision-making in an EdTech organization.

## Repository Goal

### Primary Goal

Demonstrate the complete workflow of a Business-Oriented Data Analyst solving real business problems using analytics.

### Secondary Goal

Showcase stakeholder thinking, business understanding, executive communication, and end-to-end analytics using SQL, Python, and Power BI.

## Success Criteria

This project aims to demonstrate the ability to:

- Understand business problems before analyzing data.
- Define meaningful KPIs.
- Perform end-to-end analytics.
- Build executive dashboards.
- Deliver business recommendations.
- Support stakeholder decision-making.
